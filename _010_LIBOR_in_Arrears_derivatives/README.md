**Project 10: Analytical, Monte Carlo, and Numerical Integration Pricing of LIBOR-in-Arrears Products**

This project is based on Computer Project 10. of the book The Concepts and Practice of Mathematical Finance by Mark S. Joshi, Second Edition (2008)

The project implements and benchmarks:

2. **Monte Carlo simulation**
